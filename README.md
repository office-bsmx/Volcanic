# Volcanic
 Simple and fast lua webserver written in Go

![alt text](https://github.com/office-bsmx/Volcanic/blob/main/art.jpg?raw=true)
<sub>Art generated with AI ♥️</sub>

# Building
 ```console
$ go build
```
# Running
 ```console
$ ./volcanic example
             ^ directory with main.lua 
```
